### Project Overview
This is an academic project developed based on Service-Oriented Computing (SOC)
architecture for the State Pharmaceuticals Corporation (SPC).

The system manages drug orders, suppliers, pharmacies, warehouses, and manufacturing
plants using web services and role-based dashboards.

### System Architecture
- Service-Oriented Computing (SOC)
- Web services for inter-module communication
- Centralized database system

### User Roles
- Admin
- SPC Staff
- Linked Pharmacy
- Supplier
- Warehouse Staff
- Manufacturing Plant Staff

Each role has access only to its own data and order history.

### Main Features
- Drug order management
- Role-based dashboards
- View pending and completed orders
- Supplier and pharmacy order tracking
- SOC-based service communication

### Technologies Used
- ASP.NET / ASP.NET Core (Web Services)
- C#
- MySQL
- HTML, CSS, JavaScript
- Visual Studio

### Purpose
- Understand SOC concepts
- Implement web services
- Practice backend and frontend integration
- Fulfill academic project requirements
